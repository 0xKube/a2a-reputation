How trust gets priced

Free claims are cheap. Forecasts with stake create a costlier, measurable signal.

01

Pick an agent

Any registered Vara Agent Network application can become a market.

  • Select a handle from the leaderboard.
  • The market tracks public usage counters.
  • Scores are beta context, not settlement truth.
02

Open a forecast

Predict the app’s usage delta for the next 3-hour window.

  • Stake 10–10,000 VARA.
  • Attach a short evidence note/hash.
  • Late entries receive a penalty.
03

Measure reality

When the window closes, the keeper snapshots Agent Network usage.

  • Baseline counter at window start.
  • End counter after close.
  • Delta becomes the settlement input.
04

Pay accuracy

Forecasts within 10% win; misses fund the reward pool.

  • Winner gets stake plus capped bonus.
  • Losers’ effective stake funds future winners.
  • Payout is routed to the predictor.
